# Luther Bible MCP Server

A semantic search MCP server for the German Luther Bible (1545, modern spelling), with King James Version and World English Bible included as bonus translations. Backed by OpenAI embeddings (`text-embedding-3-small`) and ChromaDB. Plugs directly into Claude Desktop.

Cross-lingual search works out of the box — query in English to find German verses, or vice versa.

---

## Project structure

```
luther-mcp/
├── index_bible.py        # Run once to build the vector DB
├── bible_mcp_server.py   # MCP server Claude Desktop connects to
├── requirements.txt
├── README.md
└── bible_chroma_db/      # Generated by indexer (gitignored)
```

## Setup

### 1. Clone the Bible database repo

```bash
git clone https://github.com/scrollmapper/bible_databases
```

Note the path to the `sqlite/` directory inside it — you'll need it below.

### 2. Install dependencies

```bash
cd C:\Projects\luther-mcp
pip install -r requirements.txt
```

### 3. Set environment variables

**Windows (PowerShell):**
```powershell
$env:OPENAI_API_KEY = "sk-..."
$env:BIBLE_DB_PATH  = "C:\path\to\bible_databases\sqlite"
$env:CHROMA_PATH    = "C:\Projects\luther-mcp\bible_chroma_db"
```

**Linux / macOS:**
```bash
export OPENAI_API_KEY="sk-..."
export BIBLE_DB_PATH="/path/to/bible_databases/sqlite"
export CHROMA_PATH="./bible_chroma_db"
```

### 4. Run the indexer (one-time setup)

```bash
python index_bible.py
```

This embeds ~93,000 verses across three translations using OpenAI `text-embedding-3-small`.

**Estimated one-time cost: ~$0.10**

Progress is printed as each batch completes:
```
[GerBoLut] 31102 verses loaded. Starting embedding...
[GerBoLut] 2048/31102 verses indexed
...
[GerBoLut] Done.
```

To re-index (e.g. after a code change):
```bash
python index_bible.py --force
```

## Add to Claude Desktop

Edit `claude_desktop_config.json` (usually at `%APPDATA%\Claude\claude_desktop_config.json`):

```json
{
  "mcpServers": {
    "Luther Bible": {
      "command": "C:\\Python313\\python.exe",
      "args": ["bible_mcp_server.py"],
      "cwd": "C:\\Projects\\luther-mcp",
      "env": {
        "OPENAI_API_KEY": "your-key-here",
        "CHROMA_PATH": "C:\\Projects\\luther-mcp\\bible_chroma_db"
      }
    }
  }
}
```

Restart Claude Desktop. The server appears under the MCP tools panel.

## Available tools

### `search_bible`
Semantic search — finds verses by meaning, not keyword.

| Parameter | Type | Default | Description |
|---|---|---|---|
| `query` | string | required | Query in any language |
| `translation` | string | `"GerBoLut"` | `"GerBoLut"` / `"KJV"` / `"web"` / `"all"` |
| `n_results` | integer | `10` | Results to return (max 50) |
| `testament` | string | — | `"OT"` or `"NT"` to filter |

### `get_verse`
Exact reference lookup by book/chapter/verse.

| Parameter | Type | Default | Description |
|---|---|---|---|
| `book` | string | required | German or English book name |
| `chapter` | integer | required | Chapter number |
| `verse` | integer | required | Verse number |
| `translation` | string | `"GerBoLut"` | `"GerBoLut"` / `"KJV"` / `"web"` / `"all"` |

### `list_translations`
Returns available translations with language, description, and verse count.

## Example queries

### German queries
- `"Gottes Liebe zur Welt"` → finds John 3:16 and similar verses
- `"Licht in der Dunkelheit"` → finds Gospel of John prologue, Genesis creation
- `"Vergebung der Sünden"` → forgiveness passages across both testaments
- `"Seid nicht ängstlich"` → "Do not be afraid" / Angst passages

### English queries (cross-lingual — returns German Luther Bible verses)
- `"God so loved the world"` → Johannes 3:16
- `"blessed are the peacemakers"` → Matthäus 5:9
- `"love your enemies"` → Matthäus 5:44
- `"the Lord is my shepherd"` → Psalmen 23:1

### Exact lookup
- Get Johannes 3:16 in Luther Bible: `get_verse("Johannes", 3, 16)`
- Get John 3:16 in all translations: `get_verse("John", 3, 16, translation="all")`
